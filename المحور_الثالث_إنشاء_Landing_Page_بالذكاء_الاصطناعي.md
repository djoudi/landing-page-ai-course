# المحور الثالث: إنشاء Landing Page بالذكاء الاصطناعي (80 دقيقة)

## الهدف

إنشاء صفحة هبوط احترافية موجهة للسوق الجزائري باستخدام أدوات الذكاء
الاصطناعي، مع نموذج تسجيل واقعي.

------------------------------------------------------------------------

## المرحلة 1: اختيار فكرة المشروع (10 دقائق)

يختار كل متدرب فكرة مثل:

-   مدرسة خاصة
-   دورة تكوينية
-   نادي روبوتيك
-   تطبيق تعليمي
-   متجر إلكتروني
-   مكتبة

------------------------------------------------------------------------

## المرحلة 2: إنشاء هوية المشروع (15 دقيقة)

### Prompt

``` text
أنت خبير Branding.

بناءً على فكرة المشروع التالية:

[اكتب الفكرة]

أنشئ:

- اسم المشروع
- شعار (Slogan)
- وصف قصير
- الجمهور المستهدف
- ثلاثة ألوان مناسبة
```

------------------------------------------------------------------------

## المرحلة 3: إنشاء محتوى الصفحة (15 دقيقة)

### Prompt

``` text
أنت خبير في كتابة صفحات الهبوط.

أنشئ Landing Page احترافية لمشروع:

[اسم المشروع]

تتضمن:

- عنوان رئيسي
- عنوان فرعي
- وصف
- خمس مزايا
- ثلاثة أسباب لاختيار المشروع
- زر "سجل الآن"
```

------------------------------------------------------------------------

## المرحلة 4: إنشاء الصفحة بالذكاء الاصطناعي (20 دقيقة)

استخدم إحدى الأدوات:

-   Lovable
-   Bolt.new
-   Framer AI

### Prompt الرئيسي

``` text
Create a modern Landing Page for an Algerian service.

The Landing Page must include:

1. Hero Section
2. Features
3. Why Choose Us
4. Registration Form
5. FAQ
6. Contact
7. Footer

Registration Form:

- Full Name
- Email
- Phone Number
- Age
- Gender
- Wilaya (Dropdown)
- Commune (Dropdown depending on Wilaya)
- Submit Button

Requirements:

- Responsive
- Mobile Friendly
- Modern UI
- Rounded Cards
- Beautiful Icons
- Soft Colors
- Smooth Animations

When the user selects a Wilaya,
show only the Communes belonging to that Wilaya.

Use Algeria administrative divisions.
```

------------------------------------------------------------------------

## المرحلة 5: تحسين التصميم (15 دقيقة)

### Prompt

``` text
راجع الصفحة كخبير UX/UI.

حسن:

- الألوان
- الخطوط
- الأزرار
- المسافات
- الصور
- سرعة القراءة

اجعل الصفحة مناسبة للأطفال والمراهقين.
```

------------------------------------------------------------------------

## المرحلة 6: اختبار الصفحة ونشرها (5 دقائق)

قائمة التحقق:

-   العنوان واضح
-   الصور مناسبة
-   الأزرار تعمل
-   نموذج التسجيل كامل
-   الولاية والبلدية تعملان
-   الصفحة متجاوبة مع الهاتف
-   لا توجد أخطاء إملائية

------------------------------------------------------------------------

# المشروع النهائي

ينشر كل متدرب صفحته ويشرح:

1.  اسم المشروع.
2.  الفئة المستهدفة.
3.  سبب اختيار التصميم.
4.  كيف يعمل نموذج التسجيل.
5.  كيف يتم اختيار الولاية والبلدية.

------------------------------------------------------------------------

# المهارات المكتسبة

-   إنشاء Landing Page.
-   كتابة Prompts.
-   استخدام أدوات AI.
-   تصميم نموذج تسجيل احترافي.
-   فهم فكرة جمع بيانات المستخدمين (Leads).
-   نشر موقع إلكتروني حقيقي.
